# Classificação de Complexidade

Simples (Haiku): edit pontual, rename, CSS, texto, typo, campo simples, boilerplate puro sem lógica
Média (Sonnet): CRUD, função com lógica, query SQL, bug multi-arquivo, endpoint, componente UI, validação
Alta (Opus): decisão arch, 5+ arquivos coordenados, debug sem causa, refactor estratégico, novo módulo com regras complexas ou multi-domínio (PRD→Spec→Código), migração, redesign schema, fluxo LGPD completo
Novo módulo CRUD simples = Média (Sonnet)

Escalamento: Média → Alta mid-task → parar e avisar antes de continuar.
